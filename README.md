# image_gather

yahoo画像検索を使って、画像を収集するツールです。
python3.6で動作確認しています。

# Installation

1. ```repository```を`clone`します。  
```$ git clone https://github.com/ta9ya/image_gather.git```

1. 必要モジュールをインストールします。  
```$ pip install -r requirements.txt```

1. 実行させます。  
```$ python image_gather.py```
